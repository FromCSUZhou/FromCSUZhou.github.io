hexo常用命令

  1. hexo n “xxxx” #新建一篇博客
  2. hexo clean #每次上传到GitHub前清除一下缓存
  3. hexo g #重新生成hexo静态文件
  4. hexo s #本地启动hexo服务
  5. hexo d #上传到github上
  6. 一般步骤为2->3->4->5
